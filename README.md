# Mohammad Adhil — Portfolio

Personal portfolio site built with React, TypeScript, Vite, GSAP, and Three.js.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deploy

Deploy to Render as a **Web Service** (not Static Site):

- **Build command:** `npm install && npm run build`
- **Start command:** `npm start`
- **Node version:** 18+

Or use the included [`render.yaml`](render.yaml) Blueprint.

For Vercel/Netlify, deploy the `dist` folder with build command `npm run build`.
